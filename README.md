<h1 align="center">TouchFish Texts</h1>

<div align="center">
  <strong>强大的 Markdown 编辑器</strong><br>
  一个简单优雅的自由开源 Markdown 编辑器。<br>
  <sub>适用于 Linux、macOS 和 Windows。</sub>
</div>


## 功能

- 实时预览 (WYSIWYG) 和简洁的界面，提供无干扰的写作体验。
- 支持 [CommonMark 规范](https://spec.commonmark.org/0.29/)、[GitHub 风格 Markdown 规范](https://github.github.com/gfm/) 以及选择性支持 [Pandoc Markdown](https://pandoc.org/MANUAL.html#pandocs-markdown)。
- Markdown 扩展功能，如数学表达式 (KaTeX)、前置内容和表情符号。
- 支持段落和内联样式快捷键，提高写作效率。
- 输出 **HTML** 和 **PDF** 文件。
- 多种主题
- 多种编辑模式
- 支持从剪贴板直接粘贴图片。
